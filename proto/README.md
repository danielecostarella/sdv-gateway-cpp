# Proto files

The `kuksa/val/v2/val.proto` file is fetched from the upstream Eclipse Kuksa
Databroker repository at CMake configure time and is intentionally not
committed here (see `.gitignore`).

To fetch it manually:

```bash
./scripts/fetch-proto.sh
```

Source: https://github.com/eclipse-kuksa/kuksa-databroker/tree/main/proto
